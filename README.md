# 개요
<>
레이캐스팅 기술을 이용하여 2차원의 맵으로 3차원의 공간을 제작하는 프로젝트입니다.
#### 레이캐스팅
- 가상의 공간에 빛을 투사해 빛에 닿는 표면의 위치를 파악하는 기술입니다. 2차원 맵을 3차원의 원근감이 있는 공간으로 보이게 하는 데에 사용됩니다. 
- 레이캐스팅을 활용한 가장 대표적인 게임은 Wolfenstein입니다.
